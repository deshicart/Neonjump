# NeonLeap - Doodle Jump Game

A neon-styled Doodle Jump clone built with Phaser 3, React, and TypeScript.

## 🎮 Features

- Classic Doodle Jump gameplay with neon aesthetics
- Multiple platform types (normal, moving, breakable, spring, boost)
- Enemy types with different behaviors
- Power-ups: Shield and Jetpack
- Sound effects and background music
- Responsive controls (keyboard and touch)
- High score tracking

## 🚀 Development

### Prerequisites

- Node.js 20 or higher
- npm

### Installation

```bash
npm install
```

### Running Locally

```bash
npm run dev
```

Visit `http://localhost:5173` in your browser.

### Building for Production

```bash
npm run build
```

The built files will be in the `dist/` folder.

## 🎯 How to Play

- **Desktop**: Use arrow keys or A/D keys to move left and right
- **Mobile**: Tap or drag on either side of the screen to move
- Jump on platforms to go higher
- Avoid enemies or use shield power-up
- Collect jetpack for a boost
- Try to beat your high score!

## 🛠️ Tech Stack

- **Game Engine**: Phaser 3
- **Framework**: React 19
- **Build Tool**: Vite
- **Language**: TypeScript
- **Styling**: Tailwind CSS
